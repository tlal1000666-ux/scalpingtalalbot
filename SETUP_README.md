# إعداد بوت توصيات التداول (تلجرام + GitHub Actions)

بوت يفحص السوق كل 30 دقيقة تلقائيًا (بدون سيرفر خاص فيك، وبدون ما جهازك يكون مفتوح)، ويرسل توصيات دخول/خروج على تلجرام حسب نفس منطق الاستراتيجية اللي اختبرناها بالباكتست بالضبط.

⚠️ **تنبيه:** هذا أداة توصيات وتتبع فقط. ما ينفذ صفقات حقيقية، وما يشكل نصيحة استثمارية. جرّبه فترة (Paper Trading) قبل أي اعتماد عليه بفلوس حقيقية.

---

## الخطوة 1: إنشاء بوت تلجرام (5 دقائق)

1. افتح تلجرام وابحث عن **@BotFather**.
2. أرسل له `/newbot` واتبع التعليمات (اسم البوت + username ينتهي بـ `bot`).
3. رح يعطيك **TOKEN** يشبه: `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` — احفظه.
4. ابحث عن بوتك بالاسم اللي اخترته وابعثله أي رسالة (مثلاً `/start`).
5. افتح هذا الرابط بالمتصفح (بدّل TOKEN بالتوكن الحقيقي):
   `https://api.telegram.org/botTOKEN/getUpdates`
6. رح تشوف رقم بحقل `"chat":{"id": 123456789 ...` — هذا هو **CHAT_ID** تبعك، احفظه.

---

## الخطوة 2: إنشاء ريبو على GitHub

1. سوّي حساب على [github.com](https://github.com) لو ما عندك.
2. اضغط **New repository** → اسمه مثلاً `trading-signal-bot` → خليه **Private** (أفضل لخصوصيتك) → Create.
3. **مهم:** لو خليته Private، تأكد إن حسابك عنده رصيد دقائق GitHub Actions كافي (2000 دقيقة/شهر مجانًا). لو تبي بدون أي حد، خليه **Public** بدل Private (Actions مجانية غير محدودة للريبوهات العامة).

---

## الخطوة 3: رفع الملفات

ارفع كل الملفات المرفقة بهذا الحل (بنفس هيكلة المجلدات) لريبو GitHub:

```
run_bot.py
strategy.py
utils.py
requirements.txt
symbols.txt
state.json
.github/workflows/signal_bot.yml
```

أسهل طريقة: اسحب كل الملفات لصفحة الريبو على GitHub.com مباشرة (زر "Add file" → "Upload files")، أو استخدم git من جهازك:

```bash
git clone https://github.com/USERNAME/trading-signal-bot.git
cd trading-signal-bot
# انسخ كل الملفات المرفقة هنا
git add .
git commit -m "إعداد أولي للبوت"
git push
```

---

## الخطوة 4: إضافة التوكن كـ Secret

1. بصفحة الريبو، روح **Settings** → **Secrets and variables** → **Actions**.
2. اضغط **New repository secret** وأضف:
   - الاسم: `TELEGRAM_BOT_TOKEN` — القيمة: التوكن من الخطوة 1
3. كرّر وأضف secret ثاني:
   - الاسم: `TELEGRAM_CHAT_ID` — القيمة: الـ chat id من الخطوة 1

---

## الخطوة 5: تفعيل الجدولة

1. روح تبويب **Actions** بالريبو.
2. لو طلب منك تفعيل Workflows، اضغط **I understand my workflows, go ahead and enable them**.
3. البوت هلأ رح يشتغل تلقائيًا كل 30 دقيقة (حسب جدول الـ cron بملف `signal_bot.yml`).
4. لتجربته فورًا بدون انتظار: روح **Actions** → اختر **Trading Signal Bot** → **Run workflow** → **Run workflow**.

---

## كيف تتأكد إنه شغّال؟

- روح تبويب **Actions** وشوف آخر تشغيل (لازم يكون بعلامة ✅ خضراء).
- اضغط على أي تشغيل وشوف الـ logs (تقدر تشوف كم رمز فحص، وشو نظام السوق العام، وأي إشارات طلعت).
- كل صفقة تُفتح أو تُغلق تنرسل مباشرة برسالة تلجرام.
- ملف `trades_log.csv` بالريبو رح يتحدّث تلقائيًا بكل صفقة مغلقة — هذا سجلك لمقارنة الأداء الحي بتوقعات الباكتست لاحقًا.

---

## تخصيص الإعدادات

- **قائمة الرموز:** عدّل `symbols.txt` (رمز بكل سطر). القائمة الحالية فيها 224 رمز (نفس اللي استخدمناها بالباكتست). لو الريبو Private وعندك حد دقائق، قلّل العدد لأهم 30-50 رمز.
- **تكرار التشغيل:** بملف `.github/workflows/signal_bot.yml`، غيّر سطر الـ cron (افتراضيًا كل 30 دقيقة).
- **حجم الصفقة / عدد الصفقات المتزامنة:** بملف `strategy.py`، غيّر `POSITION_SIZE_PCT` و`MAX_CONCURRENT_TRADES` (حاليًا 33% و3 صفقات، بحسب آخر إعداد اتفقنا عليه).

---

## ملاحظة تقنية مهمة

البوت يحفظ حالته (الصفقات المفتوحة، فترات التهدئة) بملف `state.json` ويحدّثه بنفسه بكل تشغيل عبر commit تلقائي للريبو — هذا اللي يخليه "يتذكر" الصفقات المفتوحة بين كل تشغيل وتشغيل بدون أي قاعدة بيانات خارجية.
